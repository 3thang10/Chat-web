	+ Bước 2: mở source code bằng visual studio code
	+ Bước 3: mở mongodb ấn connect ở url mongodb://localhost:27017
	+ Bước 4: mở terminal gõ cd server
	+ Bước 5: gõ npm i 
	+ Bước 6: gõ npm start

	+ Bước 9: gõ npm start
